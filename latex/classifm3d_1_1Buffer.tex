\doxysubsection{Detailed Description}
The class \mbox{\hyperlink{classifm3d_1_1Buffer}{Buffer}} represent a S\+TL conatiner to store data from the ifm devices in 2 dimension and supports multiple channel. data stores in sequnetial memory layout and class provides function template to access the pixel. Creating an \mbox{\hyperlink{classifm3d_1_1Buffer}{Buffer}} object \+: 


\begin{DoxyItemize}
\item Use the Create(cols, rows, nchannel, ifm3d\+::pixel\+\_\+format ) method or the similar Buffer(cols,rows, nchannel, type) constructor.
\end{DoxyItemize}

For example, F\+O\+R\+M\+A\+T\+\_\+8U means a 8-\/bit array, F\+O\+R\+M\+A\+T\+\_\+32F floating-\/point array, and so on.

\DoxyCodeLine{\textcolor{comment}{// and now turn image to a 10 x10 3-\/channel 8-\/bit matrix.}}
\DoxyCodeLine{\textcolor{comment}{// The old content will be deallocated}}

\end{DoxyCode}
