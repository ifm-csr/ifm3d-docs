The J\+S\+ON interface returned here is the excellent \href{https://github.com/nlohmann/json}{\texttt{ J\+S\+ON for Modern C++}}.

This function (along with its {\ttfamily std\+::string} equivalent {\ttfamily \mbox{\hyperlink{classifm3d_1_1Device_ae6a009493ff158fc3f18d86544aedad8}{To\+J\+S\+O\+N\+Str()}}}) provides the primary gateway into obtaining the current parameter settings for the device and P\+MD imager. Data returned from this function can be manipulated as a {\ttfamily json} object, then fed into {\ttfamily From\+J\+S\+O\+N(...)} to mutate parameter settings on the device.

A J\+S\+ON object representation of the current state of the hardware.
